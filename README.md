# 🏦 Crimson Desert — Timer Banque

Outil communautaire pour ne plus jamais rater le rafraîchissement
de la banque dans Crimson Desert.

## ✨ Fonctionnalités

- Calcul automatique de l'heure IRL de retour à la banque
- Heure IRL mise à jour en temps réel
- Alerte sonore + notification système 5 min avant l'échéance
- Switch mute pour couper le son
- Table de référence des conversions temps in-game / réel
- Table des niveaux de risque d'investissement

## 📐 Conversion temps

| Temps in-game | Temps réel |
|---------------|------------|
| 1 heure       | 5 min      |
| 1 jour (24h)  | 2 h        |
| 3 jours       | 6 h        |

> ⚠️ Ces valeurs sont basées sur des sources communautaires
> et peuvent être approximatives.

## 🚀 Utilisation

1. Ouvre `crimson-desert-banque.html` dans ton navigateur
2. Entre le nombre d'heures in-game affiché par le banquier
3. Clique sur **"À quelle heure revenir ?"**
4. Laisse la page ouverte sur ton 2ème écran
